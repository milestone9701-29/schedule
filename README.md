# Schedule API

> 3-Layer + JPA + MySQL + JWT.


## Roadmap (D1~D3)
- D1: domain(User/Schedule/Comment), - repository - dto - controller - service
- D2: 예외, JWT
- D3: idempotencyKey 실습, update의 경우, @Version 응용하여 if문 분기처리.

## Run
- JDK 17, Gradle, MySQL 8
- `application.yml` 작성 후 `gradlew bootRun`
